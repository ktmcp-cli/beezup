# @ktmcp/beezup

MCP server for BeezUP Merchant API 

## Installation

```bash
npm install -g @ktmcp/beezup
```

## Usage

Add to your MCP settings:

```json
{
  "mcpServers": {
    "beezup": {
      "command": "mcp-beezup"
    }
  }
}
```
